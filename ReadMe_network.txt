#NetworkShortPath.py

AUTHOR: Harry Hamilton-Deeley

DESCRIPTION:
This program is designed to find the shortest/cheapest route through a network.

USAGE:
The program is controlled by changing the input file (inputnetwork.txt).

Enter the cost for each link in the input file. Each line should have the costs from the top to the bottom.
Please only enter the links without any spaces.

Example:

2,1
2,3,3,2
2,3,6,2,4,5
3,4,5,1,2,3,3,4

NOTE: If the table in the log file (lognetwork.txt) does not allign, open it directly in notepad.
